## Tổng quan

Trang **Settings** cho phép bạn:

- Quản lý thông tin tài khoản
- Tùy chỉnh giao diện (theme)
- Quản lý phòng ban
- Quản lý chức vụ

---

## Truy cập

Vào **Settings** từ sidebar menu.

---

## Account Settings

### Thông tin tài khoản

Hiển thị thông tin user hiện tại:

```
┌─────────────────────────────────────────────┐
│ Tài khoản                                   │
├─────────────────────────────────────────────┤
│                                             │
│ Email                                       │
│ admin@company.com                           │
│                                             │
│ Vai trò                                     │
│ Administrator                               │
│                                             │
│ Ngày tạo                                    │
│ 01/01/2024                                  │
│                                             │
└─────────────────────────────────────────────┘
```

### Đổi mật khẩu

1. Vào **Settings** → **Account**
2. Click **Đổi mật khẩu**
3. Nhập mật khẩu cũ và mới
4. Lưu

---

## Appearance (Giao diện)

### Theme

Chọn giao diện hiển thị:

```
┌─────────────────────────────────────────────┐
│ Giao diện                                   │
├─────────────────────────────────────────────┤
│                                             │
│ Theme                                       │
│ ○ Light    ○ Dark    ● System              │
│                                             │
│ Font                                        │
│ [Default ▼]                                 │
│                                             │
└─────────────────────────────────────────────┘
```

| Theme      | Mô tả                 |
| ---------- | --------------------- |
| **Light**  | Giao diện sáng        |
| **Dark**   | Giao diện tối         |
| **System** | Theo cài đặt hệ thống |

### Font

Chọn font hiển thị:

- Default
- Sans-serif
- Serif
- Mono

## Quản lý Phòng ban

### Truy cập

Settings → **Departments**

### Danh sách phòng ban

```
┌─────────────────────────────────────────────┐
│ Phòng ban                          [+ Thêm] │
├─────────────────────────────────────────────┤
│ # │ Tên           │ Số nhân viên │ Thao tác│
├───┼────────────────┼──────────────┼─────────┤
│ 1 │ IT             │ 15           │ ⋮       │
│ 2 │ HR             │ 8            │ ⋮       │
│ 3 │ Finance        │ 12           │ ⋮       │
│ 4 │ Marketing      │ 6            │ ⋮       │
└───┴────────────────┴──────────────┴─────────┘
```

### Thêm phòng ban mới

1. Click **+ Thêm**
2. Nhập tên phòng ban
3. Lưu

### Sửa/Xóa phòng ban

Click menu **⋮** để:

- **Sửa:** Thay đổi tên
- **Xóa:** Xóa phòng ban (nếu không có nhân viên)

> ⚠️ Không thể xóa phòng ban đang có nhân viên.

## Quản lý Chức vụ

### Truy cập

Settings → **Positions**

### Danh sách chức vụ

```
┌─────────────────────────────────────────────┐
│ Chức vụ                            [+ Thêm] │
├─────────────────────────────────────────────┤
│ # │ Tên           │ Số nhân viên │ Thao tác│
├───┼────────────────┼──────────────┼─────────┤
│ 1 │ Developer      │ 10           │ ⋮       │
│ 2 │ Manager        │ 5            │ ⋮       │
│ 3 │ Accountant     │ 4            │ ⋮       │
│ 4 │ Designer       │ 3            │ ⋮       │
└───┴────────────────┴──────────────┴─────────┘
```

### Thêm chức vụ mới

1. Click **+ Thêm**
2. Nhập tên chức vụ
3. Lưu

### Sửa/Xóa chức vụ

Click menu **⋮** để:

- **Sửa:** Thay đổi tên
- **Xóa:** Xóa chức vụ (nếu không có nhân viên)

> ⚠️ Không thể xóa chức vụ đang được sử dụng.

---

## Tips & Best Practices

### Phòng ban

- ✅ Tạo phòng ban trước khi tạo nhân viên
- ✅ Đặt tên ngắn gọn, dễ hiểu
- ✅ Sắp xếp theo alphabetical order

### Chức vụ

- ✅ Tạo các chức vụ cần thiết trước
- ✅ Đặt tên chuẩn hóa (ví dụ: Developer, Manager)
- ✅ Không tạo quá nhiều chức vụ không cần thiết

### Giao diện

- ✅ Sử dụng Dark mode nếu làm việc nhiều giờ
- ✅ Để System để tự động theo cài đặt OS
